# Factory Placement MVP — каркас

Этот репозиторий содержит каркас проекта по кейсу:
**«Умный подбор локации и дизайна для производства с местным характером»**.

## Архитектура
- **Frontend:** Streamlit + Folium (карта + форма инвестора)
- **Backend:** FastAPI (эндпоинты ранжирования, расчета площадей/сметы, генерации отчетов)
- **Данные:** локальная JSON база регионов/площадок (для стабиль��ой демо)

## MVP (план)
1) Форма инвестора (10 полей)
2) Расчет площадей и сметы
3) Ранжирование ТОП‑3 регионов на основе весов критериев
4) Карта с выбором площадки
5) Генерация аналитической справки (Markdown) — сначала шаблон, затем LLM
